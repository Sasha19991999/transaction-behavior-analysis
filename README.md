# Transaction Behavior and Customer Value Analysis

## Objective
This project analyzes an e-commerce transaction dataset to understand customer behavior, product performance, and revenue trends.

## Dataset
The dataset contains transactional data including:
- CustomerID
- StockCode
- Product Description
- Quantity
- Unit Price
- Date and Time
- Country

## Key Analysis Performed
- Revenue performance analysis
- Customer segmentation (Revenue-based tiers)
- Product performance analysis
- Time-based behavioral analysis
- Identification of peak sales periods

## Key Insights
- Revenue peaks at 12 PM, indicating highest purchasing activity during midday.
- Thursday generates the highest revenue, showing strong midweek demand.
- Revenue increases significantly toward the end of the year.
- A small number of products generate a large proportion of total revenue.
- High-value customers contribute disproportionately to overall revenue.
